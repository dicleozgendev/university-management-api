# University Management API

A layered university management system developed with ASP.NET Core following clean architecture principles.

## Technologies

- ASP.NET Core Web API
- C#
- Entity Framework Core
- SQL Server
- Repository Pattern
- Dependency Injection

## Architecture

```
Presentation
    ↓
Business
    ↓
DataAccess
    ↓
Entity
```

## Features

- Student Management
- Teacher Management
- Course Management
- RESTful API
- Layered Architecture

## Project Structure

- University.API
- University.Business
- University.DataAccess
- University.Entity

## Getting Started

```bash
git clone https://github.com/dicleozgendev/university-management-api.git
```

Open the solution in Visual Studio and run the API.
